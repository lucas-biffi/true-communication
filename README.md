# TrueCommunication - Plataforma de Relatos de Golpes Bancários

## Sobre o Projeto
Este projeto foi proposto na universidade e tem como objetivo oferecer uma plataforma colaborativa onde usuários possam registrar e consultar relatos de golpes bancários recebidos por telefone.  
A finalidade deste projeto é criar uma base de dados aberta que ajude a identificar padrões, números suspeitos e aumentar a conscientização.

## Funcionalidades
- Registro de relatos de golpes recebidos por ligação telefônica.
- Consulta de relatos cadastrados
- Sistema de categorização (tipo de golpe, número de telefone, banco mencionado e descrição)  
- Tela principal com dicas de segurança

## Tecnologias Utilizadas
- **Backend:** Node.js / Express / MySQL2 / CORS
- **Frontend:** Html / CSS / JavaScript  
- **Banco de Dados:** MySQL Workbench
